## Correspondence to Rust Source

```rust
// quiche/src/frame.rs:1272–1311 (simplified)
let smallest_ack = largest_ack - ack_block;
let mut ranges = vec![(smallest_ack, largest_ack)];
let mut smallest = smallest_ack;
for (gap, blk) in blocks {
    if smallest < 2 + gap { return Err(…) }
    let lg = (smallest - gap) - 2;
    if lg < blk { return Err(…) }
    let sm = lg - blk;
    ranges.push((sm, lg));
    smallest = sm;
}
```

The Lean model `decodeAckBlocks` is a direct functional translation of
this loop with identical guard conditions and arithmetic.
